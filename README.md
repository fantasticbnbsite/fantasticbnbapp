# Fantastic BNB MVP

Aplicacao web multi-clientes para substituir planilhas por uma operacao centralizada e escalavel.

## Como rodar

1. Abra um terminal em `C:\Users\vini3\Desktop\GRAVIDADE\APP FANTASTIC`
2. Execute `node server.js`
3. Acesse `http://localhost:3000`

## Logins de demo

- `admin@fantasticbnb.app` / `admin123`
- `operacoes@fantasticbnb.app` / `manager123`
- `viewer@fantasticbnb.app` / `viewer123`

## Recursos do MVP

- Multi-clientes com acesso segmentado
- Login com sessoes e permissoes
- CRUD de registros
- Tabela estilo planilha com busca, filtros e ordenacao
- Campos e status customizados por cliente
- Historico de alteracoes
- Importacao CSV com criacao automatica de campos
- Exportacao CSV
- Backup automatico local do banco SQLite
- API pronta para evoluir para dashboards, financeiro e integracoes
